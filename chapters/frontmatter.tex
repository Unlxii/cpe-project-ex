\begin{abstractTH}
VisScan เป็นระบบตรวจสอบและวิเคราะห์ความปลอดภัยของซอร์สโค้ดและอิมเมจคอนเทนเนอร์ที่พัฒนาขึ้นเพื่อเสริมศักยภาพด้าน DevSecOps ภายในแพลตฟอร์ม VisOperation ระบบนี้ออกแบบมาเพื่อช่วยให้องค์กรสามารถตรวจจับช่องโหว่ ความผิดพลาดในการตั้งค่า และความเสี่ยงด้านความปลอดภัยที่อาจเกิดขึ้นในกระบวนการพัฒนาและปรับใช้แอปพลิเคชันได้อย่างรวดเร็วและแม่นยำ รองรับการสแกนซอร์สโค้ด การตรวจสอบ Secret Leakage การสแกนอิมเมจ Docker และการประเมินความปลอดภัยของคอนเทนเนอร์โดยใช้เครื่องมือมาตรฐานอุตสาหกรรม เช่น Trivy, Gitleaks และ Static Analysis Engine ต่าง ๆ อีกทั้งยังมีระบบจัดการผลการสแกน การแจ้งเตือน และการเชื่อมต่อเข้ากับ Pipeline อัตโนมัติ เพื่อให้ทีมพัฒนาและทีมปฏิบัติการสามารถดำเนินงานได้อย่างราบรื่นและปลอดภัยยิ่งขึ้น ด้วยการทำงานร่วมกับ VisOperation ทำให้ VisScan กลายเป็นส่วนประกอบสำคัญของโครงสร้างพื้นฐานด้านความปลอดภัย ช่วยเพิ่มความโปร่งใสในการตรวจสอบ ลดความเสี่ยงด้านความปลอดภัย และสนับสนุนการตัดสินใจเชิงปฏิบัติการขององค์กรได้อย่างมีประสิทธิภาพ
\end{abstractTH}


\begin{abstract}
VisScan is a security analysis system for source code and container images, developed to enhance DevSecOps capabilities within the VisOperation platform. Designed to assist organizations in rapidly and accurately detecting vulnerabilities, misconfigurations, and potential security risks during application development and deployment, the system supports source code scanning, secret leakage detection, Docker image scanning, and container security assessment using industry-standard tools such as Trivy, Gitleaks, and various Static Analysis Engines. Additionally, it features scan result management, notifications, and automated pipeline integration, facilitating smoother and more secure operations for development and operations teams. By integrating with VisOperation, VisScan becomes a crucial component of security infrastructure, increasing audit transparency, reducing security risks, and effectively supporting operational decision-making.
\end{abstract}
